# MERN Stack E-commerce Platform

Welcome to the MERN Stack E-commerce Platform! This web application is built using the MERN (MongoDB, Express.js, React.js, Node.js) stack, providing a robust and scalable solution for online shopping.

## Features

- **Frontend**: Developed with React.js to ensure a responsive and interactive user interface. Users can seamlessly navigate through the platform, view products, and make purchases.

- **Payment Integration**: Secure payment transactions are enabled through PayPal, providing a reliable and widely-used payment gateway for users.

- **Backend**: Powered by Node.js and Express.js, the backend ensures efficient data handling and communication between the server and the frontend.

- **Database**: MongoDB is used as the database, ensuring a flexible and scalable storage solution for user data, product details, and other essential information.

- **User Authentication**: The platform offers a secure authentication system that includes features such as user registration, login, and a forgot password functionality.

- **Product Management**: Users can explore products through various categories, utilize filters, and search functionality to find specific items. Admins have the ability to add or delete categories and manage product details.

- **User Management**: Admins have control over user details, allowing them to make necessary changes and maintain the user database.

## Getting Started

To run the project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone(https://github.com/ft-prince/prince-ecommerce).git
   ```

2. Navigate to the project directory:

   ```bash
   cd mern-ecommerce-platform
   ```

3. Install dependencies for both frontend and backend:

   ```bash
   cd client
   npm install
   cd ../server
   npm install
   ```

4. Configure environment variables:

   Create a `.env` file in the `server` directory and provide necessary configuration, including MongoDB connection string and PayPal API credentials.

5. Run the project:

   ```bash
   # In the server directory
   npm start

   # In the client directory
   npm start
   ```

   The application will be accessible at `http://localhost:3000/`.
